## Best practice

1. basic (must do)
    1. 1st glance
        1. pronounce
        2. word frequency
        3. categorized meaning list
        4. mean frequency
    2. 2nd glance
        1. associated words
         2. synonyms for each meaning
         3. antonyms
         4. word family
         5. derivatives
    3. variants
         1. complex
         2. tense
         3. comparative
    4. fast understanding
        1. simple description ([Collins](https://www.collinsdictionary.com/dictionary/english-thesaurus) style)
        2. example sentences for each mean
        3. compositional phrasemes
2. advanced (optional and inefficient)
    1. meaning of paraphrase
    2. idioms
        1. weak idiom
        2. semi-idiom
        3. full idiom
    3. morpheme
        1. root
        2. affix
    4. etymology
